Screening Task: Power Systems Simulation using OpenModelica and OpenIPSL

  1. By using the OpenIPSL library in OpenModelica, Create one to three models. You can submit any number of models maximum being three and the screening would be done based on the complexity of the models.
  2. Take any power system network (from a project report, websites, textbook or a journal) and model it in OpenModelica using OpenIPSL library
  3. Please make sure that the submitted simulations are not under the Completed Power System Simulations.
